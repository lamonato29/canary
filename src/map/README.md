# Map

This directory contains classes and functions for managing the game map.

## Key Components

- `Map`: Represents the game world, managing tiles, floors, and pathfinding.
- `Tile`: Represents a single tile on the map, holding items and creatures.
- `House`: Manages houses and their tiles.
- `Town`: Manages towns and temple positions.
- `Spawn`: Handles monster and NPC spawning.
