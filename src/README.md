# Source Code

This directory contains the source code for the Canary server.

## Subdirectories

- `game/`: Contains the core game logic, including map, creatures, movement, and game mechanics.
- `items/`: Contains item definitions, properties, and interactions.
- `creatures/`: Contains creature definitions (players, monsters, NPCs) and their behaviors.
- `io/`: Contains input/output operations, file loading, and network handling.
- `lua/`: Contains Lua scripting integration.
- `database/`: Contains database connection and query handling.
- `config/`: Contains configuration loading and management.
- `map/`: Contains map data structures and loading.
- `server/`: Contains server startup and main loop.
- `utils/`: Contains utility functions and helper classes.
