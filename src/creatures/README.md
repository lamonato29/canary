# Creatures

This directory contains the definitions and logic for all living entities in the game.

## Key Components

- `Creature`: The base class for all moving entities (Players, Monsters, NPCs).
- `Players`: Contains player-specific logic, including vocations, skills, and inventory.
- `Monsters`: Contains monster logic, AI, and spawn management.
- `NPCs`: Contains NPC logic, trading, and interactions.
- `Combat`: Handles combat formulas, spells, and conditions.
